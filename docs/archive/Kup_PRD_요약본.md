# Kup — PRD (요약본 · 제출용)

> 1~2장 요약 PRD. 개발자가 읽으면 "이런 서비스구나"가 바로 잡히는 것이 목적. 상세 스펙은 별도 `Kup_SPEC.md` 참조.

## 1. 한 줄 정의

인스타를 처음 시작하는 사람이 매일 콘텐츠를 직접 만들지 않고, **주제·타깃만 고르면** AI 자동화가 카드뉴스·릴스를 매일 제작·발행하고 **코멘트→DM**으로 팔로워를 모아, 가장 어려운 **첫 1,000명(수익화 진입선)**까지 도달한다. — *성장 우선, 1K 이후 수익화.*

## 2. 타깃 · JTBD

**민준(34) 등 예비 창작자**: "내 주제로 수익화할 인스타 계정을 만들고 싶은데, 매일 콘텐츠 만들 시간·기획력이 없어서, 운영을 자동으로 맡기고 한 달 안에 지쳐 포기하지 않고 1,000명을 넘고 싶다."

타깃 3종 → 한 엔진의 3가지 모드로 커버
- **A** 금융·부동산 관심 2030 (지식 O · 소스 ✗) → *생성형 모드*
- **B** 자기 브랜드 소상공인 (목표 = 고객 유입·매출) → *자산 기반 모드*
- **C** SPA 패션 2030 (영상 O · 편집 ✗) → *증강 모드*

## 3. 핵심 Pain ↔ Reliever

- 초반 무반응에 지쳐 한 달 내 포기 → 자동 운영 + 진척 트래커로 **콜드스타트(0→1K) 통과**
- 매일 기획·디자인·발행 피로 → 주제만 고르면 **매일 자동 제작·발행**
- "뭘 올리지" 기획력 격차 → **주제 추천·탐색**(핫 / 팔로워 잘 모이는 / 조회수 높은) + 전략 자동 생성
- 팔로워가 안 모임 → **코멘트→DM 리드마그넷**(공식 API)으로 참여·리드 동시 확보
- 알고리즘 불투명 → 성과 데이터 기반 인사이트·훅 추천

## 4. MVP 핵심 기능

**① 자동 콘텐츠 운영 (주 기능)** — 주제·타깃 선택 + 추천·탐색 → 전략·카드뉴스(템플릿+브랜드 컬러)·릴스 스크립트 자동 생성 → **승인 후 발행(검수 기본 ON)** → n8n 예약 발행. *카드뉴스 메인 + 릴스 보조.*

**② 팔로워 수집 + 성과 (Aha)** — 코멘트→DM 리드마그넷, 성과 대시보드. **첫 자동 발행("내가 안 했는데 올라갔다")이 Aha moment.**

*엔진 = 3모드:* 생성형(A) / 자산 기반(B) / 증강(C·영상→숏츠)

## 5. 안전장치 · 차별화

- **휴먼 인 더 루프 검수 기본 ON + AI 생성 고지 라벨** → 품질·신뢰 *(상세 설계는 SPEC 부록 참조)*
- **공식 API만 사용**(자동팔로우·콜드DM 금지) → 계정 정지 리스크 차단
- 주제→전략→제작→발행→리드까지 **엔드투엔드** (단일 툴들과 차별)
- 도메인 컴플라이언스: 금융=권유 표현 차단 / 패션·소상공인=협찬·뒷광고 자동 표기
- *시장 인지:* 인스타는 2026년 자동화·템플릿 콘텐츠 도달을 낮추는 추세 → 위 휴먼인루프 + AI라벨 + 코멘트→DM + 니치 전략으로 정면 대응

## 6. 기술 구조 · 핵심 결정

- **데이터:** 사용자 IG 비즈니스 계정 **OAuth 연결**(비번 미보관·토큰 암호화), 콘텐츠·성과는 Meta Graph API 수집
- **자동화:** n8n 오케스트레이션(예약·발행·웹훅) + 자체 스케줄러 + 템플릿 렌더
- **AI:** Claude API = 기획·카피·스크립트 등 **언어 가공만** / 영상은 ZapCap·Creatomate 등
- **3-레이어 분리(핵심):** 콘텐츠 생성 = Claude / 인스타 발행·데이터 = Meta Graph API(앱 검수 필수) / 오케스트레이션 = n8n. *(Claude API로는 인스타 직접 접근 불가)*

## 7. 범위 · Out of Scope

- **범위:** 카드뉴스 메인 + 릴스 보조 · 생성형/자산 기반 모드 · 코멘트→DM · **한국 우선**
- **Out (이후):** 자동팔로우·대량/콜드 DM·가짜 인게이지먼트 · "1K 보장" 마케팅 · 수익 정산 대행 · 숏츠 완전 생성(증강 모드 C)은 M2 · 멀티 계정 · 글로벌·다국어

## 8. 성공 기준 (KPI)

**미션 지표(장기·가치 실현):** 🌟 1,000 팔로워 도달 계정 수 *(주간 운영 관리엔 미사용)*

**운영 North Star(주간):** ⭐ 주간 활성 채널당 팔로워 순증
- 분해: `팔로워 순증 ≈ 발행 수 × 게시물당 도달 × (도달→팔로우 전환율)`
- 마일스톤(가설): 1개월 +100 / 3개월 누적 500 / 6개월 1,000

**서비스 데이터(퍼널):** 가입자 수 · IG 연동 완료율(70%+) · 활성화율(48h 내 첫 승인·발행, 40%+) · **2회차 발행률(7일 내, 가설 60%)** · 5건 자동발행 후 코호트 잔존율 · W4 발행 유지율(25%+)

**인스타 데이터(Graph API):** 게시물당 유입 대비 이탈(순증 효율) · 주간 리드마그넷 유입

**가드레일(동시 감시):** 게시물당 참여율(필수 짝) · 무수정 승인율(60~85%) · 평균 검수 시간(<1~2분) · 계정 제재·셰도우밴율(0) · 고위험 검수 누락률(0%)

**가치 증명:** 연속 발행 회원 vs 중단 회원의 1K 도달 속도 비교 = 성장 리프트
