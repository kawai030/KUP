## 3. 카드뉴스 템플릿
- **6종 템플릿**: 표지형 · 리스트형 · 비교형 · 인용형 · 강조형 · CTA형
- **태그**: 전 템플릿 공통, 헤드라인 위. 숫자 입력 → 번호 뱃지 / 글자(AI 등) → 칩
- **강조형**(구 통계형): 단위·캡션 입력칸 제거 → 큰 숫자 하나 + 본문이 헤드라인 밑에
- **미디어 배치**: 반반(위 미디어/아래 글) · 배경+DIM. 비율 1:1 / 3:4
- **화이트·블랙 테마**에선 태그를 핑크 대신 글자색(검정/흰색)으로
- 텍스트 전용 vs 미디어 유무에 따른 폰트 크기 스케일

## 4. 워크스페이스 craft 상향 (전 화면)
- **카드 소프트 엘리베이션** — 테두리만 있던 플랫 → 종이 위에 뜬 느낌
- **버튼 프레스 피드백** + 핑크 브랜드 그림자
- **입력 포커스** 회색 → 브랜드 핑크 링
- **빈 상태** 회색 박스 → 따뜻한 핑크 아이콘 뱃지(맥락별 아이콘)
- **클릭 행 hover** 마이크로인터랙션 — 핑크 틴트 + 화살표 슬라이드
- **홈 '최근 콘텐츠'**: 색상 블록 → **카드 첫 장 미니 미리보기**(실제 렌더)
- 적용 화면: 홈 · 제작(에디터) · 콘텐츠 관리 · 성과 · DM · 기획 · 계정 관리

## 5. 랜딩
- **히어로**: 카드 수천 장이 3D 구(球)로 뭉치는 스크롤 애니메이션 (카드→채널, 브랜드 은유)
- **히어로 텍스트 가독성**: 소프트 화이트 스크림 + 페이드 타이밍 조정 (구와 겹쳐 안 읽히던 문제 해결)
- **가짜 성과 지표 제거** (`+12% · 70% · 12,000+`) → 실제 기능 앵커(말투 학습 · 예약 발행 · 성과 정리) + 정직 안내문
  - PRD **"가짜 지표 금지"** 원칙 준수 — 지어낸 수치 대신 인스타 연동 후 실계정 데이터만 노출
